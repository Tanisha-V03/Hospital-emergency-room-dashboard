# Hospital Emergency Room Dashboard

## Project Overview

This project is an interactive Hospital Emergency Room Dashboard developed in Microsoft Excel. It provides a comprehensive analysis of emergency room operations by tracking patient volume, waiting time, admissions, referrals, patient satisfaction, and demographic information. The dashboard is designed to help users monitor key performance indicators and support data-driven decision-making.

## Project Objective

The objective of this project is to analyze hospital emergency room data and present meaningful insights through an interactive Excel dashboard. The dashboard enables users to monitor operational performance, identify trends, and improve healthcare management.

## Dashboard Features

- Total Number of Patients
- Average Wait Time
- Patient Satisfaction Score
- Admission Status Analysis
- Patient Attendance (On Time vs Delayed)
- Patient Distribution by Gender
- Patient Distribution by Age Group
- Department Referral Analysis
- Interactive Month and Year Filters

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Excel Formulas
- Conditional Formatting
- Data Cleaning
- Data Visualization

## Key Insights

- Analyzed monthly patient trends.
- Compared admitted and non-admitted patients.
- Evaluated average waiting time.
- Measured patient satisfaction scores.
- Analyzed patient distribution by age and gender.
- Examined department referral patterns.
- Tracked on-time and delayed patient attendance.
